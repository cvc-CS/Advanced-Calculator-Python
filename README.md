# Advanced-Calculator-Python
As my first Python project, I asked myself: what's something everyday students like me could actually use? That's how this scientific calculator came to the top of my mind. Built with Python &amp; tkinter, featuring a custom dark UI, memory functions, and trig/log operations.


# Python Scientific Calculator

A GUI calculator built with Python and tkinter, styled with a modern dark theme.

## Features
- Basic arithmetic (+, -, ×, ÷)
- Scientific functions (sin, cos, tan, log, ln, √, x², factorial)
- Memory functions (M+, M-, MR, MC)
- Keyboard input support
- Calculation history display
- Custom UI with hover/press micro-interactions

## How to run
1. Make sure Python is installed
2. Run: `python Calculator.py`

## What I learned
Event-driven programming, GUI design with tkinter, error handling, writing reusable functions.
